The floating action button gives Slime a page-level control for quick form filling without opening the toolbar popup.

## Enable It

Open the options page and configure the action button. You can enable or disable it, adjust its position, and set the keyboard shortcut.

## Default Rules

Assign default rules to URL patterns so one click can run the right fill profile on pages you test often.

## Shortcut

The default shortcut is `Ctrl+Q`. Change it if that conflicts with the website or another extension.
