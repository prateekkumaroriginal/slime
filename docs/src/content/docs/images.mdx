Slime can store image files in extension storage and attach them to file inputs when a matching rule runs.

## Add Images

Open the options page, go to image storage, and add the image assets you use for testing. Slime supports common browser image formats such as PNG, JPEG, GIF, and WebP.

## Use An Image In A Rule

1. Create or edit a rule.
2. Add a field mapping for a file input.
3. Choose the image value type.
4. Select one of the stored images.
5. Save and run the rule on a matching page.

## Storage

Images use Chrome extension storage. Keep the image library focused so the extension remains fast to back up and restore.
